Data Structures

This repository contains *Data Structure programs developed for practice and learning.  
The programs are implemented using C and Java and focus on building strong logical thinking, problem-solving skills, and understanding core data structure concepts*.

Contents
- Arrays  
- Linked Lists  
- Stack  
- Queue  
- Searching Techniques  
- Basic Algorithms  
- Logical Practice Programs

 How to Compile and Run
  For C language
  gcc filename.c -o Myexe 
  for linux : ./Myexe for Windows : Myexe.exe

  For C++
  g++ filename.cpp -o output 
  for linux : ./Myexe for Windows : Myexe.exe

  For Java
  javac filename.java 
  java filename OR java filename.java
